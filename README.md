# CollegeChatbotRAG
Chatbot for your college. This Streamlit app is a multi-functional dashboard with:  
a) A chat interface  
b) A section to add notifications  
c) A section to upload files 


This application integrates various technologies to provide a seamless experience for users to interact with PDF documents, authenticate securely, and obtain answers to their questions using advanced language models.
